= NOWCODER_HUAWEI_ANS

牛客网华为机试题答案，目前有c++版

== 目录说明

- /Cpp

存放C++版的试题答案，每道题目有对应的一个.cpp文件。

- /Tools

存放astyle配置文件，用于格式化C++的代码。

- /.vscode

存放vscode编辑器的配置文件，由vscode自动生成。

== 代码风格

使用astyle管理代码风格，配置文件为Tools/astyle/astylerc。bash环境下参考命令：

[source,bash]
----
$ astyle --options=./Tools/astyle/astylerc --recursive ./path_to_file/*.c,*.h
----
